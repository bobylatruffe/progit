Comment initialiser un dépot git ?
	git init 

Quelle commande permet de savoir l'état des fichiers dans un dépot git ?
	git status

Qu'est-ce que sa veut dire qu'un fichier est non-suivi ?
	Que git detecte que ce fichier n'est pas présent dans le dernier instantané.

Comment placer des fichiers sous suivi de version ?
	git add nom_fichier

Si un fichier a déjà été indexé, et qu'on le modifie, qu'est-ce qu'il faudra faire ?
	Il faut le re-ajouter à l'index 
	git add nom_fichier

Comment lancer la cmd git status de manière courte ?
	git status -s | --short

Que signifie :
$ git status -s
 M README
MM Rakefile
A lib/git.rb
M lib/simplegit.rb
?? LICENSE.txt

	README a été modifier et indexé
	Rakefile a été modifier et indexé, mais que des modification plus récente que l'index existe
	git.rb est un nouveau fichier et indexé
	simplegit.rb a été modifier et indexé
	LICENCE.txt est un nouveau fichier et non suivi

Comment faire pour ignorer tous un dossier particulier dans dossier ?
	dans .gitignore :
		/**/build/


Peut-on avoir d'autre fichier .gitignore pour un dépôt ?
	Oui, il est possible d'avoir des fichiers .gitignore pour chaque sous dossier d'un dépôt.


Comment afficher ce qui a été modifié mais pas encore indexé concrêtement ?
	git diff
		Cette cmd permet de comparer l'espace de taf avec celui dans l'indexe.

Comment comparer ce qui est dans l'indexe par rapport au dernier intantané ?
	git diff --cached | --staged

Comment utiliser faire lancer un outil extern au lieu de git diff ?
	git config --global diff.tool "outils"
	git difftool

Comment paramétrer l'editeur à utiliser quand on fait un git commit sans l'option -m ?
	git config --global core.editor "editeur"

Comment faire un commit de tous ce qui a été modifier et déjà suivi par git sans l'ajouter à l'index ?
	git commit -a

Comment supprimer un fichier en suivi de version ?
	git rm 
	git rm -f pour forcer un fichier modifier et indexé 

Comment abandonné le suivi d'un fichier tout en le conservant dans l'espace de taf ? (oublie de le signaler dans .gitignore)
	git rm --cached fichier

Comment afficher avec un git log les différence introduite entre chaque commit ?
	git log -p

Si je me suis trompé dans le message d'un commit, comment faire pour modifier le message du commit ?
	git commit --amend

Si j'ai déjà commit, mais que je me rend compte que j'ai oublié d'indexé un fichier, qu'est-ce que je fais ?
	git add fichier_oublié
	git commit --amend

Si j'ai un fichier modifié et indexé mais que je veux le desindexé ?
Si j'ai un fichier modifié mais non indexé et que je veux récupérer les modifications du dernier commit pour se fichier ?
	Normalement après git status, git indique comment faire, dans ma version actuelle  c'est : 
		git restore --staged nom_fichier

		git restore nom_fichier
