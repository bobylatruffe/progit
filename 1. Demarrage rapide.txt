Les logiciels de versionning autre que Git enregistre les modifications de fichiers, 
alors que Git procède comment ?
	Git enregistre des instantanés lors d'un commit.
	C'est un peu comme si chaque enregistrement d'un projet dans Git était un système de fichier à part entière.

Dans Git, un fichier peut se trouver dans différents états, citez-les sans rentrer dans le details ?
	- modifié
	- indexé
	- validé

Comment afficher toutes les config de git (system, global) ?
	git config --list --show-orgin

Comment changer l'editeur de texte de git ?
	git config --global core.editor nom_unix | chemin

Comment changer le nom de la branche principale avec la cmd git config ?
	git config --global init.defaultBranch nom
